# CalmMindsetCo — The Calm Money System

A free, beautiful budgeting web app for people who find money stressful.

## Project Structure
```
calmmindsetco/
├── public/
│   └── index.html      ← Main site (all-in-one)
├── vercel.json          ← Vercel config
└── README.md
```

## Deploying updates
1. Edit `public/index.html`
2. Push to GitHub
3. Vercel auto-deploys in ~30 seconds
